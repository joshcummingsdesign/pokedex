# Pokedex

A demo project coded by voice.

For information on voice, coding, check out the [vocalize](https://github.com/joshcummingsdesign/vocalize) project.

## Getting started

1. Start the api

   ```sh
   cd api
   pip install -r requirements.txt
   make start
   ```

2. Start the ui

   ```sh
   cd ui
   npm install
   npm run dev
   ```
